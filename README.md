# TruSynth
Simple single-voice subtractive synthesizer written with JUCE &amp; C++

## Features
- 2 oscillators
  - Sine, square & sawtooth waveforms
  - Adjustable level, pitch & fine-tuning controls
  
- Low pass filter
  - Adjustable cutoff threshold
  - Resonance control

## Builds with
- JUCE v6.0.8
- Visual Studio/C++ 2019
